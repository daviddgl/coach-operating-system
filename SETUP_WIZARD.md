# Setup Wizard (Coach OS)

- Name, niche, current model (1:1, group, hybrid)
- Primary coaching outcomes

## Phase 2 — Kernel
- Coaching purpose statement
- Core values
- Pressure mode indicators
- Decision protocol preferences

## Phase 3 — Practice Config
- Practice mission and vision
- Quarterly strategy and North Star
- Service offerings

## Phase 4 — Portfolio Drivers
- Active clients/cohorts
- Capacity contract (coaching / biz dev / admin)
- Referral network

## Phase 5 — Processes
- Quarterly goals and milestones
- Key risks and mitigations

## Phase 6 — Command + Boardroom
- Preferred command cadence (daily/weekly)
- Advisory persona customization

Output: populated COS files ready for AI-assisted use.
